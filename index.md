2. **Select the Version**  
   On the Releases page, look for the latest version of FoodSwift.

3. **Download the Zip File**  
   Click on the zip file link to download. This will contain all the necessary files to run the application.

4. **Extract the Zip File**  
   Once downloaded, locate the zip file in your downloads folder. Right-click on the file and select "Extract All" to unpack it. You will see a folder named "FoodSwift" after extraction.

5. **Install Dependencies**
   Open a terminal or command prompt:
   
   - **For Windows:** Press `Windows + R`, type `cmd`, and hit `Enter`.
   - **For macOS/Linux:** Open the Terminal app.

   Navigate to the extracted folder using `cd` command. For example:
   ```
   cd path/to/FoodSwift
   ```
   Then run the following command to compile dependencies:
   ```
   mvn clean install
   ```

6. **Run the Application**  
   Once the installation is complete, navigate to the `target` directory inside the FoodSwift folder. Execute the application by running:
   ```
   java -jar FoodSwift.jar
   ```

7. **Access FoodSwift**  
   Open your web browser and type in `http://localhost:8080` to launch FoodSwift.
